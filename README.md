# 📚 Online Book Store SQL Project

A beginner-friendly SQL project built using **PostgreSQL** to practice database design and data analysis using an online bookstore dataset.

This project was created as part of my SQL learning journey to strengthen my understanding of relational databases and solve real-world business questions using SQL.

---

## 🎯 Project Objective

- Practice SQL on a real-world dataset
- Learn database design and relationships
- Perform data analysis using SQL queries
- Build a beginner portfolio project

---

## 🛠️ SQL Concepts Used

- SELECT, WHERE
- ORDER BY, GROUP BY, HAVING
- Aggregate Functions (SUM, AVG, COUNT)
- INNER JOIN & LEFT JOIN
- DISTINCT
- LIMIT
- COALESCE

---

## 📊 Business Questions Solved

- Calculate total revenue
- Find the top customer
- Identify the most ordered book
- Analyze books sold by genre
- Calculate remaining stock
- Find repeat customers
